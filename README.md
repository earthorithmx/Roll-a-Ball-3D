# 🎮 Roll-a-Ball 3D Game

## 📌 Description
This is a 3D Unity game where the player controls a ball to collect objects and score points.

---

## ✨ Features
- 🎮 Ball movement using physics
- 🧱 Object collection system
- 🧮 Score tracking
- 💀 Game over condition

---

## 🛠️ Built With
- Unity Engine
- C#
